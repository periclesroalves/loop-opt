Polly basics:
-------------

- Configure LLVM/Polly w/ assert, wo/ optmize:
  Dist files are stored under build/Debug+Asserts
  NOTE: other build can be put under this same build directory.
${LLVM_SRC}/configure --with-cloog=${CLOOG_INSTALL} --with-isl=${CLOOG_INSTALL} --enable-libcpp=yes

- Running Polly - lib load + vector code generation:
alias pollycc="./clang -Xclang -load -Xclang ~/ufmg-research/build/tools/polly/Debug+Asserts/lib/LLVMPolly.dylib -O3 -mllvm -polly"

- Get the IR before Polly optimization:
./clang -Xclang -load -Xclang ~/ufmg-research/build/tools/polly/Debug+Asserts/lib/LLVMPolly.dylib -O0 -mllvm -polly -S -emit-llvm

- Generate PDF of detected SCoPs (also shows alias info):
on Clang: -polly-show
on opt: -view-scops
